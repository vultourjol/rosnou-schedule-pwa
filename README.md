# РОСНОУ Расписание PWA

PWA-приложение для просмотра расписания занятий Российского Нового Университета.

## Возможности

- Установка как приложение на телефон/компьютер
- Загрузка расписания из Excel-файла
- Удобный календарь с отображением занятий
- Тёмная тема
- Сохранение расписания в браузере
- Работа оффлайн

## Использование

1. Перейдите на [страницу расписаний](https://rosnou.ru/student/shedule)
2. Найдите свою группу и откройте Excel-файл
3. Скачайте файл: **Файл → Создать копию → Скачать копию**
4. Нажмите кнопку «Загрузить Excel» и выберите скачанный файл

## Установка на устройство

### iOS (Safari)
1. Откройте сайт в Safari
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой)
3. Выберите «На экран Домой»

### Android (Chrome)
1. Откройте сайт в Chrome
2. Нажмите меню (три точки)
3. Выберите «Установить приложение»

### Компьютер (Chrome/Edge)
1. Откройте сайт
2. Нажмите иконку установки в адресной строке
